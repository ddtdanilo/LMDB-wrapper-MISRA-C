# vcpkg Port Reference

This directory contains a **reference copy** of the vcpkg port files for `ddtdanilo-lmdb-wrapper`.

The authoritative version lives in the [microsoft/vcpkg](https://github.com/microsoft/vcpkg) registry at `ports/ddtdanilo-lmdb-wrapper/`.

These files are kept here for development reference and to track changes between versions. When releasing a new version, update these files and submit a PR to the vcpkg registry.
